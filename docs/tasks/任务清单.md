## 技术栈更新要点

### 前端技术栈
- **开发框架**: 微信小程序原生开发
- **UI组件库**: TDesign MiniProgram UI组件库
- **状态管理**: app.js + getApp()全局数据管理
- **网络请求**: 基于wx.request封装
- **Canvas绘制**: Canvas 2D API

### 后端技术栈
- **数据库**: MySQL 8.0+
- **ORM**: SQLAlchemy with MySQL connector
- **认证**: JWT Token认证
- **API**: FastAPI RESTful接口
